# Chuck Norris Client

Next.js + TypeScript client app for Chuck Norris jokes.

## Requirements implemented

- Load 10 random jokes on page load
- Rolling list (max 10) with a toggleable 5s timer
- Mark jokes as favorites (max 10)
- Favorites page with remove
- Favorites persist across refresh (localStorage)
- CI pipeline + tests

## Tech stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- Vitest + Testing Library
- ESLint + Prettier
- Husky + lint-staged
- GitHub Actions + React Doctor

## Getting started

### Node version

Use Node 22 (see `.nvmrc`).

### Install

```bash
npm install
```
